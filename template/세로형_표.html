세로형 표 열 추가, 행 추가

세로형 표 데이터를 추가할 때 현재 html 태그의 특성상 CSS 클래스만으로 조작할 수 없습니다.

또한 현재의 표에서 어디까지 추가할 수 있는 지 확인 후 명시해놓을테니

데이터 상 칸이 몇개까지는 대응되어야 하는지에 대해 기준과 한계점에 대해 조율이 필요합니다.




* 태그관련 설명 : tr : 테이블의 행을 의미, th: 테이블의 제목을 의미 , td: 테이블의 내용을 의미 행을 추가하려면 tr, th , td를 순서대로 추가해야합니다.

다음 테이블의 경우 가로형 테이블과 달리 colgroup에 col 태그도 먼저 추가를 해주어야 합니다. 자세한 방법은 아래 주석을 참고하세요.




데이터 행을 추가하는 방법 :

1. colgroup안 col 태그 복사 붙여넣기 (열 간격 조정 및 설정)

2. thead 태그안의 th 태그 추가하기 (제목 추가)

3. tbody 태그안의 td 태그 추가하기 (내용 추가)

4. 데이터 추가가 많아서 표가 망가질 경우 col tag 속성 조정
